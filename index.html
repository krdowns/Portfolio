A little about me...

"We have to continually be jumping off cliffs and developing our wings on the way down"

- Kurt Vonnegut -

My name is Kenny Downs and I am a front-end web developer in San Francisco. I am from Atlanta, but just over a year ago, I decided to take the leap to move to San Francisco. Whether it is in music, gaming, sports, or code, I love meaningful and creative interactions. My work includes both front-end and full stack web and mobile sites.

Currently, I love working in HTML, CSS, JavaScript, and Node.js. I have found that creating a great user experience, whether on my own or as part of a team with designers, is one of the most gratifying aspects of web development. Please take a look at my work, and feel free to contact me with any questions.

My Work

Mobile-Only

See

See is a mobile-only web app designed as an outlet for users who struggle with mental illness. Using Watson's Tone Analyzer API, it analyzes the emotions of a body of text and shows the user a visual breakdown of the overall history of their entries.

Hiking Meetup

Hiking Meetup is a mobile-only web app designed as an introduction to the company and as a way for a user to signup for a newsletter to find out about future Meetups and see past Meetup highlights.

Responsive

Monument

Monument is a responsive travel blog template which was built using HTML and CSS.

CRWN Clothing Concept Project (in progress)

CRWN Clothing is a concept project built using React and Redux on the front end and Google Firebase on the backend. It also uses Stripe's API for payments. It was originally styled with SCSS but then transitioned to using styled components.
